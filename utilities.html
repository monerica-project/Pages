  • ✅ XMR.ID - Accept Monero payments at yourname@xmr.id

  • XMR.IST - OpenAlias service, convert long wallet addresses to human readable format. Try monerochan@xmr.ist in your wallet.

Blockchain Explorers

  • 3xpl - Block explorer

  • Blockchair - Search for transactions

  • Explore Monero - Verify transactions and explore the network

  • Monero Blocks - Monero blockchain blocks

  • Monero Consensus Status - Visualization of recent orphaned blocks and alternative chains of the Monero blockchain.

  • monerowatch - Beautiful Monero Explorer (Note: JS required)

  • xmrchain.net - Monero explorer

  • Xmrscan - A Monero block explorer.

Node Lists

  • monero.fail - Mainnet, stagenet and testnet Monero nodes for clearnet web, TOR and I2P

  • Stealthxmr.io - Monero Node with no tracking, no logs, just a privacy driven node.

  • XMR Nodes - Public Monero remote nodes list

Open Source Projects

  • Docker Monero Node - A docker-compose project to package a Monero node with an advanced monitoring dashboard.

  • Easynode - A software simplifying the node deployment process. (Note: Monero Node)

  • EmbassyOS Wrapper for Monero - Wraps the Monero daemon for EmbassyOS.

  • ETH-XMR Atomic Swaps - A an implementation of ETH-XMR atomic swaps, currently in beta.

  • Live XMR wishlist as a service

  • Memonero - A monero brain wallet.

  • Mitra - Self-hosted microblogging server with integrated Monero payments

  • Monero Brain Wallet Generator - Generate a monero brain wallet.

  • Monero Discourse Subscriptions Plugin - This plugin enables you to give your users access to private Discourse groups and categories for Monero payments.

  • Monero Ecosystem - Collection of Monero projects managed by members of the community

  • Monero GitHub Project List - hundehausen's list of Monero GitHub Projects

  • Monero Node for Umbrel - Run a Monero node on your Umbrel personal server.
